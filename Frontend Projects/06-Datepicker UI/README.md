# Datepicker UI

基于 [roadmap.sh Datepicker UI 项目](https://roadmap.sh/projects/datepicker-ui) 的练习实现。

## 项目目标
- 使用 HTML + CSS 创建一个美观、响应式、可访问的日期选择器UI组件
- 实现完整的日历界面，包含月份导航、日期网格、操作按钮等
- 熟悉现代CSS布局技术和日期选择器设计最佳实践

## 页面与布局
- 单页面组件，采用居中布局，顶部有标题，下方为日期选择器
- 日期选择器包含输入框、日历图标、下拉日历弹出层
- 日历弹出层包含月份导航、星期标题、日期网格、操作按钮
- 响应式设计，适配不同屏幕尺寸
- 暗色模式支持，自动适配系统主题

### 主要分区
- Datepicker UI 标题
- 日期选择器组件（输入框 + 日历弹出层）

## 结构与体验优化
- 语义化 HTML 标签（如 `<div>`, `<button>`, `<input>`, `<span>` 等）
- `<head>` 包含 SEO meta 标签（title、description、keywords、author、robots）
- 使用 CSS Grid 实现日期网格布局，Flexbox 实现组件对齐
- **暗色模式**：自动适配系统主题
- **可访问性**：ARIA标签、键盘导航、焦点管理、减少动画偏好支持
- **动画与细节**：悬停效果、过渡动画、毛玻璃效果、SVG图标

### 组件结构
- **输入区域**：文本输入框 + 日历图标按钮
- **日历弹出层**：月份导航 + 星期标题 + 日期网格 + 操作按钮
- **日期状态**：当前月日期、上月/下月日期、今天日期、选中日期

### 交互设计特点
- **悬停触发**：鼠标悬停时自动显示日历弹出层
- **视觉反馈**：按钮和日期悬停时的状态变化
- **状态指示**：不同日期类型使用不同颜色和样式
- **操作便捷**：今天和清除按钮提供快捷操作

### 响应式设计
- **桌面端 (480px+)**: 标准布局，日历弹出层相对定位
- **移动端 (<480px)**: 日历弹出层居中显示，适配小屏幕

## 如何使用
1. 直接在浏览器中打开 `index.html` 预览效果
2. 将鼠标悬停在日期输入框上查看日历弹出层
3. 可根据需要修改日期内容和样式
